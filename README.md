## Запуск тестов

```bash
# Все тесты
./gradlew test

# Только математика
./gradlew test --tests "com.arekalov.tpolab1.math.*"

# Только красно-черное дерево
./gradlew test --tests "com.arekalov.tpolab1.rbtree.*"

# Только вселенная
./gradlew test --tests "com.arekalov.tpolab1.universe.*"

# С покрытием
./gradlew test jacocoTestReport
```

## Демонстрация доменной модели

Запустите пример использования доменной модели:

```bash
./gradlew run
```

Пример демонстрирует:
1. **Основной сценарий** - полный жизненный цикл от дискуссий до получения ответа "42"
2. **Брокианский ультра-крикет** - игра как способ снятия стресса
3. **Путь знания** - развитие через размышления
